How should users pick an animal template?

The template picker lets kids choose a built-in line-art background (fox, panda, cat, dog…). Where and how should this appear?

A

Bottom Sheet / Modal Grid

Tapping a "Templates" button in the toolbar slides up a full-width panel showing animal thumbnails in a scrollable grid. User taps one to load it, then the sheet closes.

Feels like: iOS photo picker or emoji keyboard
B

Horizontal Scroll Strip

A row of animal thumbnail cards sits permanently in the bottom panel (between brushes and palette). Tap one to make it the active template. Always visible, no extra tap needed.

Feels like: Sticker row in iMessage
C

Dedicated Template Screen

A separate "Choose Template" screen (navigated from the toolbar) shows all animals in a large grid. User selects one and is taken back to the canvas.

Feels like: Sticker pack store in WhatsApp
